# The Setting
You are summoned to a world filled with Darkness. In such a way that you can only see other people talk as text on terminal and things as ASCII art and such. You are honestly baffled. A friendly voice(text...actually) begs you to bring their world back to what it used to be, where there was light and people talked in their own voices and there was music and ambience in the world. You must find the Person responsible for the state of this world, so that you can save it and just return to some sembelence of sanity.

Good Luck Hero! Survive in this hellish terminal only landscape and bring peace to this world!
